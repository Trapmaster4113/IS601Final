Dockerhub: https://hub.docker.com/r/tlc23/is601final <br/>

To Install: 
1. Clone github repository/Download repository
2. Create Python Virtual Environment (python -m venv venv)
3. Activate Python Virtual Environment (source venv/bin/activate)
4. Install Python Packages (pip install -r requirements.txt)
5. Install Playwright dependencies (playwright install) <br/>
To Build:
1. docker compose up -v <br/>

Testing:
- After installation, run "pytest"
- Test logs stored in htmlcov
- Test files stored in directory: "tests" <br/>

Added Features: <br/>
1. New Calculation: Exponent
